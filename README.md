# Philosophers

In this project, you will learn the basics of threading a process.

You will see how to create threads and you will discover mutexes.

_____
This is a classical task about the dining philosophers:
https://ru.wikipedia.org/wiki/Задача_об_обедающих_философах

### How to use

Follow next steps:
1) make
2) ./philo number_philosophers   time_to_die     time_to_eat    time_to_sleep (e.g. "./philo 4 500 200 200")
